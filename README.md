📁 Aplicación de Gestión de Proyectos
Aplicación web para gestionar proyectos de forma sencilla. Los usuarios pueden subir y administrar sus proyectos, mientras que los administradores cuentan con un panel completo para gestionar usuarios, proyectos y configuraciones.

🚀 Funcionalidades
Registro e inicio de sesión

Subida y gestión de proyectos

Panel de administración

Roles: usuario y administrador

Interfaz simple e intuitiva

🛠️ Tecnologías
Frontend: (HTML, CSS, JavaScript)

Backend: (Node, PHP, Python…)

Base de datos: (MySQL)
